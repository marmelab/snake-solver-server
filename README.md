Snake Solver Server
===

Snake Solver Server is a server for [snake game](https://github.com/marmelab/snake_solver_two).

Install
---

`go get -u github.com/marmelab/snake-solver-server`

Run
---

- `snake-solver-server`
- go to http://127.0.0.1:1323

Test
---

- `make install`
- `make test`
